# bd-skills

Agent skills for the [bitdrift Capture](https://bitdrift.ai) mobile observability platform.

## Skills

- **bitdrift-instrumentation** — SDK integration: custom logs, fields, crash reporting, debugging
- **bitdrift-platform** — Platform operations: workflows, alerts, bd tail, session timelines

## Quick start

Paste this into any AI coding agent:

> Set up bitdrift in your project. Follow https://bitdrift.ai/agent-setup
